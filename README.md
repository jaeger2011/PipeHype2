Name: PipeHype
OS: Android 

PypeHype ist eine Android App, die eine Lautstärkemessung über das integrierte Mikrofon durchführt und das Ergebnis auf dem Display grafisch darstellt. Das Ganze ist in ein Mini-Spiel verpackt, in welchem der Spieler in die Rolle eines männlichen Vogels schlüpft und es darum geht durch einen möglichst langen und lauten Pfeifton möglichst viele "Vogeldamen" anzulocken.
